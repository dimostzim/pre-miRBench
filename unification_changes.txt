--- CHANGE 1: mire2e ---
File changed: tools/mire2e/inference.py

What was changed:
  - Replaced JSON output (predictions.json) with CSV output (predictions.csv).
  - Sub-window aggregation (previously done in tool_adapters.py) moved inline:
    mire2e internally re-windows each input record; the new code strips the
    trailing -<start>-<end> suffix from each sub-window index to recover the
    parent record_id, then takes max(score_5_3, score_3_5) across all sub-windows
    belonging to that record.
  - One row per original FASTA input record.
  - Removed json import; added csv import.
